# Agent 规范

## 项目一句话

该项目采用LangGraph编排、分层内存进行上下文记忆处理，并引入一些具有确定性机制的战斗引擎，优化幻觉影响的多智能体LLM文本角色扮演框架。基于FastAPI + React。

## 必须完成的规范

- 始终使用中文回复。
- 修改代码或文档后，在 `CHANGE_LOG.md` 记录时间、执行模型、变更类型、涉及文件、变更内容和验证结果。
- 尊重分层边界
- 新增公开 API 必须补齐文档。
